# Product Images for Stripe

These HTML files contain product icons for each service package. Use them to generate PNG images for Stripe product listings.

## Icons Created

1. **quick-wins-package.html** - Lightning bolt (speed/quick results)
2. **full-audit-optimization.html** - Checkmark on document (thorough audit)
3. **custom-development.html** - Grid/layout symbol (website building)
4. **monthly-retainer.html** - Clock (ongoing/recurring service)

## How to Generate PNG Images for Stripe

### Method 1: Screenshot (Easiest)

1. Open each HTML file in your browser
2. Take a screenshot (Cmd+Shift+4 on Mac, then spacebar to capture window)
3. Crop to square if needed
4. Save as PNG

### Method 2: Use Browser DevTools

1. Open the HTML file in Chrome/Edge
2. Press F12 to open DevTools
3. Click the three dots menu → More tools → Run command (Cmd+Shift+P)
4. Type "Capture screenshot" → select "Capture node screenshot"
5. Click on the `<body>` element in Elements tab
6. Screenshot will download automatically

### Method 3: Online Converter (Recommended for perfect 1200x1200)

1. Go to https://cloudconvert.com/html-to-png
2. Upload each HTML file
3. Set output size to 1200x1200px
4. Convert and download

## Uploading to Stripe

1. Log in to Stripe Dashboard
2. Go to Products → Select your product
3. Click "Add images"
4. Upload the corresponding PNG file
5. Recommended size: 1200x1200px minimum (Stripe will auto-resize)
